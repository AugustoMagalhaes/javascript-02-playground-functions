## Projeto: Playground Functions

O segundo projeto realizado na trybe teve como objetivo testar nossos conhecimentos em Javascript, abordando os seguintes temas:
- Uso de variáveis e tipos primitivos.
- Conceitos como tipagem dinâmica e operadores lógicos/aritmétiicos.
- Estruturas condicionais (if/else, ternário, etc).
- Manipulação de arrays/listas e objetos.
- Utilização de 'loops for' e 'for/in'.
- Uso de funções pra quebrar problemas complexos em etapas menores, assim como fazer um bom uso da lógica de programação.
